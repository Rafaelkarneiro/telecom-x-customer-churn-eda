# telecom-x-customer-churn-eda
Projeto de análise de evasão de clientes da Telecom X. Os dados são coletados via API e tratados com técnicas de ETL em Python. Inclui limpeza, transformação, análise exploratória (EDA) e visualizações estratégicas para identificar padrões de churn e gerar insights que apoiem modelos preditivos e estratégias de retenção.
